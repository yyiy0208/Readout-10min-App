# Readout-10min-App 页面设计文档

## 1. 整体设计风格
- 简洁、清晰的设计风格
- 舒适的阅读体验优先
- 柔和的色彩方案，减少视觉疲劳
- 统一的设计语言和交互模式

## 2. 网页端界面设计

### 2.1 文件管理页

#### 2.1.1 核心功能
- 文件上传
- 文件列表展示
- 文件管理操作

#### 2.1.2 布局结构
```
+-------------------------------------+
| 顶部导航栏                           |
+-------------------------------------+
|                                     |
|        文件上传区域                  |
|  [选择文件] [上传]                  |
|                                     |
+-------------------------------------+
|                                     |
|        文件列表                     |
|  +-------------------------------+  |
|  | 文件名    | 类型 | 大小 | 操作 |  |
|  +-------------------------------+  |
|  | file1.pdf | PDF | 2MB  | 解析 |  |
|  +-------------------------------+  |
|  | file2.docx| DOCX| 1MB  | 解析 |  |
|  +-------------------------------+  |
|                                     |
+-------------------------------------+
```

#### 2.1.3 交互设计
- 点击"选择文件"按钮打开文件选择对话框
- 支持拖拽文件到上传区域

- 点击"解析"按钮进入文件处理页

## 4. 移动端组件设计

### 4.1 卡片组件
- 圆角设计：8px
- 阴影效果：轻微阴影，增强层次感
- 内边距：16px
- 边框：1px 浅色边框

### 4.2 按钮组件
- 主按钮：蓝色背景，白色文字
- 次要按钮：浅灰色背景，深色文字
- 圆角设计：4px
- 点击效果：轻微缩放动画

### 4.3 文本样式
- 标题：18px，加粗
- 正文：16px，常规
- 辅助文字：14px，浅色
- 朗读文本：可调整，默认18px

### 4.4 色彩方案
- 主色调：蓝色 (#4285F4)
- 背景色：白色 (#FFFFFF)
- 夜间模式：深灰色 (#121212)
- 文本色：黑色 (#000000)，夜间模式白色 (#FFFFFF)
- 辅助色：浅灰色 (#F5F5F5)
- 强调色：绿色 (#34A853)，用于成功状态

## 5. 响应式设计

### 5.1 网页端响应式
- 桌面端：双栏布局
- 平板端：自适应布局，必要时转为单栏
- 移动端：单栏布局，简化操作

### 5.2 移动端适配
- 支持不同屏幕尺寸：4.7英寸到7英寸
- 适配横竖屏模式
- 支持不同分辨率
