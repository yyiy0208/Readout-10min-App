# Readout-10min-App 页面设计文档

## 1. 整体设计风格
- 简洁、清晰的设计风格
- 舒适的阅读体验优先
- 柔和的粉色系配色方案，减少视觉疲劳
  - 主色调：粉色 (#ff8ba7)
  - 次要色：浅粉色 (#ffc6c7)
  - 第三色：薄荷绿 (#c3f0ca)
  - 背景色：白色 (#fffffe)
  - 卡片背景：浅米色 (#faeee7)
  - 标题色：深棕色 (#33272a)
  - 正文色：中棕色 (#594a4e)
- 统一的设计语言和交互模式
- Ant Design风格的组件设计

## 2. 网页端界面设计

### 2.1 文件管理页

#### 2.1.1 核心功能
- 文件上传（支持点击或拖拽）
- 文件列表展示
- 文件管理操作（解析、删除）
- 自动解析功能
- 账号管理

#### 2.1.2 布局结构
```
+-------------------------------------+
| 顶部导航栏                           |
| [文件管理] [文件处理] [账号管理 ▼]  |
+-------------------------------------+
|                                     |
|        文件上传区域（Ant Design风格）|
|  📁 点击或拖拽文件到上传区域          |
|  支持单个或批量上传                  |
|                                     |
+-------------------------------------+
|                                     |
|        文件列表（表格布局）          |
|  +-------------------------------+  |
|  | 文件名    | 类型 | 大小 | 操作 |  |
|  +-------------------------------+  |
|  | file1.pdf | PDF | 2MB  | 分段 |  删除 ||
|  +-------------------------------+  |
|  | file2.docx| DOCX| 1MB  | 分段 |  删除 ｜|
|  +-------------------------------+  |
|                                     |
+-------------------------------------+
```

#### 2.1.3 交互设计
- 点击上传区域或拖拽文件触发文件选择
- 上传完成后自动开始解析
- 点击"解析"按钮进入文件处理页
- 点击"删除"按钮删除文件（粉色主题色按钮）
- 表格行无悬停动画，偶数行有浅色背景
- 账号管理按钮点击展开下拉菜单，包含退出功能
- 点击页面其他位置关闭下拉菜单

### 2.2 文件处理页

#### 2.2.1 核心功能
- 文件内容预览
- 段落拆分设置（仅目标时长）
- 拆分结果预览

#### 2.2.2 布局结构（上下布局）

|  +---------------------------------+|
|  | [拆分] [重置] [保存]            | |
|  +---------------------------------+|
|                                     |
+-------------------------------------+
|                                     |
|        文件内容预览                 |
|  +---------------------------------+|
|  | 文件内容预览 | 文件信息         | |
|  +---------------------------------+|
|  | 文章标题和内容                   | |
|  +---------------------------------+|
|                                     |
+-------------------------------------+
|                                     |
|        拆分结果                     |
|  +---------------------------------+|
|  | 拆分结果 | 总段落数：X          | |
|  +---------------------------------+|
|  | 段落1：文本内容                 | |
|  +---------------------------------+|

#### 2.2.3 交互设计
- 拖动滑块调整目标时长
- 点击"拆分"按钮生成拆分结果
- 点击"保存"按钮保存拆分结果（粉色主题色按钮）
- 按钮水平排列，简洁明了
- 拆分结果仅显示段落数和文本内容，无时长信息
- 段落标题使用深色，与背景形成对比
- 导航栏与文件管理页一致，包含账号管理下拉菜单
- 点击"返回"按钮回到文件管理页

## 4. 移动端组件设计

### 4.1 卡片组件
- 圆角设计：8px
- 阴影效果：轻微阴影，增强层次感
- 内边距：16px
- 边框：1px 浅色边框

### 4.2 按钮组件
- 主按钮：粉色背景 (#ff8ba7)，白色文字
- 次要按钮：浅粉色背景 (#ffc6c7)，深棕色文字 (#33272a)
- 圆角设计：8px
- 点击效果：轻微缩放动画

### 4.3 文本样式
- 标题：18px，加粗，深棕色 (#33272a)
- 正文：16px，常规，中棕色 (#594a4e)
- 辅助文字：14px，浅色，中棕色 (#594a4e)
- 朗读文本：可调整，默认18px

### 4.4 色彩方案
- 主色调：粉色 (#ff8ba7)
- 次要色：浅粉色 (#ffc6c7)
- 第三色：薄荷绿 (#c3f0ca)
- 背景色：白色 (#fffffe)
- 卡片背景：浅米色 (#faeee7)
- 夜间模式：深灰色 (#121212)
- 文本色：深棕色 (#33272a)，夜间模式白色 (#FFFFFF)
- 辅助色：浅米色 (#faeee7)
- 强调色：薄荷绿 (#c3f0ca)，用于成功状态

## 5. 响应式设计

### 5.1 网页端响应式
- 桌面端：双栏布局
- 平板端：自适应布局，必要时转为单栏
- 移动端：单栏布局，简化操作

### 5.2 移动端适配
- 支持不同屏幕尺寸：4.7英寸到7英寸
- 适配横竖屏模式
- 支持不同分辨率
