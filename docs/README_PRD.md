# Readout-10min-App 产品需求文档（PRD）

## 1. 产品概述

### 1.1 产品定位
Readout-10min-App是一款包含网页端和安卓移动端的英语口语练习应用，帮助用户每天通过10-15分钟朗读练习，建立英语语感和肌肉记忆。

### 1.2 解决的核心问题
1. **多格式文件阅读不便**：用户的英文学习资料多为PDF、Word、TXT格式，在手机上查看不便
2. **段落拆分困难**：难以将长文档拆分为适合10-15分钟朗读的段落
3. **练习记录缺失**：无法有效跟踪阅读进度和历史记录

### 1.3 系统架构
- **网页端**：负责文件上传、解析、智能段落拆分和内容管理
- **安卓移动端**：负责朗读练习、进度跟踪和练习记录管理

## 2. 用户需求

### 2.1 目标用户
希望通过日常朗读提升英语口语能力的学习者，尤其是：
- 希望培养英语学习习惯的初学者
- 需要保持英语水平的职场人士
- 准备英语考试的学生

### 2.2 用户痛点
1. 缺乏系统化的朗读练习方法
2. 纸质或电子文档在手机上阅读体验差
3. 无法准确控制每次练习的时长
4. 没有有效的进度跟踪机制

## 3. 功能需求

### 3.1 网页端功能

#### 3.1.1 文件上传与管理
- 支持上传PDF、Word（.docx）、TXT格式文件
- 已上传文件列表管理（查看、删除）
- 文件分类/搜索功能

#### 3.1.2 文件解析
- 提取文本内容
- 保留基本格式（段落结构、标题层次）
- 处理特殊字符和格式
- 生成可用于移动端的纯文本格式

#### 3.1.3 智能段落拆分
- 基于预估朗读时间（10-15分钟/段）自动拆分
- 确保在句子结束处拆分，保持语义完整性
- 允许用户调整每段的目标时长（可自定义）
- 拆分结果预览功能

#### 3.1.4 内容同步与管理
- 将处理后的文本内容同步到云端数据库
- 生成可供移动端访问的唯一内容ID
- 已处理内容列表管理

### 3.2 安卓移动端功能

#### 3.2.1 内容获取与管理
- 从云端获取已处理的文本内容列表
- 支持内容下载与离线访问
- 内容分类与搜索

#### 3.2.2 朗读练习界面
- **舒适阅读体验**：
  - 可调整字体大小、行间距
  - 夜间模式支持
  - 全屏阅读模式
  - 背景色调整

- **朗读辅助功能**：
  - 自动滚动（可调节速度）
  - 段落标记（已读/未读）
  - 暂停/继续功能
  - 语速参考（显示预估朗读时长）

#### 3.2.3 练习记录管理
- **阅读进度跟踪**：
  - 当前阅读位置自动保存
  - 每篇文章的阅读进度（已读百分比）
  - 最近阅读历史

- **练习统计**：
  - 累计练习天数
  - 总练习时长
  - 每日练习完成情况


### 3.3 后续扩展功能

#### 3.3.1 网页端扩展
- 批量文件上传与处理
- 内容编辑功能
- 用户账号系统

#### 3.3.2 安卓移动端扩展
- 朗读示范（标准发音播放）
- AI朗读评估
- 生词标记与学习
